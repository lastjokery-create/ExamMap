# ExamMap v2.0 alpha

資格試験の日程、学習記録、得点推移、合否、取得済み資格を一元管理する Android アプリです。

## 実装済み

- 正式日・月のみ・上旬・中旬・下旬・未定に対応した試験日管理
- 残日数、手動進捗、結果発表後の合否確認
- 試験の登録・編集・アーカイブ・復元
- 過去問得点の登録・編集・削除・並べ替え
- 合格済み資格一覧
- 端末内への永続保存と旧データの読み込み
- Android Studioで開けるGradleプロジェクト構成

## 起動

Android Studio Ladybug以降でこのフォルダを開き、JDK 17でSync後、`app`を実行してください。minSdkは26です。

## 次期実装候補

科目別得点、集計除外、JSONバックアップ、ホーム画面ウィジェット、PC入力APIはデータ移行を伴うため、v2系で段階的に追加します。
